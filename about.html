About Marginalia

Marginalia is a small collection of text and developer tools built to answer one question quickly, without an account, a download, or a tracker following you to the next tab.

What's here

A word counter, a case converter, a JSON formatter, Base64 and URL encoding, a password generator, a slug generator, a Lorem Ipsum generator, a regex tester, a color converter, a Unix timestamp converter, and a line-based text diff checker. Each one runs entirely in your browser.

How it works

Nothing you type is sent to a server. Every tool on this site runs as JavaScript in your own browser tab, so your text never leaves your device.

Why it's free

This site is supported by ads. They help keep the tools free and available without asking for your email address.

Who this is for

Marginalia is built for the small, repetitive text tasks that come up constantly in ordinary work: a developer checking whether a JSON response is well-formed, a writer confirming an article hits a word count, a marketer building a clean URL slug for a new page, or anyone who just needs a quick, disposable password. None of these tasks need a full application — they need a fast answer and nothing else.

Design philosophy

Every tool here is intentionally single-purpose. Rather than one bloated "everything tool" with a cluttered interface, each utility does one job well, loads instantly, and gets out of the way once you have your answer. New tools get added over time as it becomes clear they'd genuinely help — not to pad out a list.

Margin notes

Each tool has a small note field underneath it — an actual margin to write in, in keeping with the name. Anything typed there is saved to your own browser only, using local storage, and reappears the next time you visit that tool on the same device. It's meant for the kind of thing you'd otherwise forget: which project a generated password belongs to, or a reminder of why a particular regex pattern works.

The sidebar's "Recently used" list works the same way — it just remembers which tools you've opened on this device, so the ones you reach for most are one click away next time.
